# Retail Sales Analysis

## Project Overview
This project analyzes retail sales data from a superstore to identify top products, customers, and profit trends. The analysis is done using Python (pandas, matplotlib, seaborn) and visualized using Power BI dashboards.

## Key Features
- Data cleaning and preprocessing
- Top products and sales analysis
- Customer segmentation and analysis
- Profit margin by category
- Sales trends over time
- Interactive Power BI dashboard

## Dataset
- Source: Superstore.csv
- 9994 records, 21 columns including Sales, Profit, Product, Customer, and Region.
